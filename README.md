# Maximus Precision Starter

Proyecto inicial en SwiftUI para crear cotizaciones, generar PDF y compartirlo por WhatsApp o cualquier app compatible con el share sheet de iPhone.

## Qué incluye
- Formulario de cliente y vehículo
- Lista dinámica de refacciones y mano de obra
- Plantillas rápidas para conceptos frecuentes
- Generación de PDF
- Vista previa del PDF
- Compartir PDF

## Cómo abrirlo
1. Descarga y descomprime el zip.
2. Abre `MaximusPrecision.xcodeproj` en Xcode 15 o superior.
3. Ejecuta en simulador o iPhone.
